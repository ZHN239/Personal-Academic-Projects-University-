A simple interactive game developed using Processing. The player controls a ball with the mouse, avoiding multiple moving balls while attempting to change the color of all flags by touching them. This project explores basic graphics rendering, collision detection, and simple game logic implementation.



Processing を用いて開発したシンプルなインタラクティブゲームです。プレイヤーはマウスでボールを操作し、画面上を動く複数のボールを避けながら、すべての旗に触れて色を変えることを目標とします。本プロジェクトでは、基本的な描画処理、当たり判定、およびシンプルなゲームロジックの実装を学びました。



一个基于 Processing 开发的简单交互式小游戏。玩家通过鼠标控制小球，在躲避屏幕上随机运动的其他小球的同时，触碰所有旗帜使其变色以完成目标。该项目主要用于探索基础的图形绘制、碰撞检测以及简单的游戏逻辑实现。
