\# 🧮 Arithmetic-Runner (Academic, Team Project)



\## 📌 Overview



Arithmetic Runner is a collaborative game project developed using Pygame.

The game combines basic arithmetic operations with real-time movement mechanics, where the player must make optimal decisions to maximize their score.



\---



\## 🎮 Gameplay



\* The player controls a character that can move left and right

\* Walls containing arithmetic expressions approach from the front

\* Each wall presents \*\*four different expressions\*\*

\* The player must choose the best option to maximize their value

\* Difficulty increases as the game progresses



\---



\## 🚀 Features



\* Real-time character movement (left/right)

\* Randomly generated arithmetic expressions

\* Multiple expressions displayed simultaneously (4 options)

\* Progressive difficulty with level-up system

\* Collision-based selection mechanics

\* Scene transitions (Main Menu / Gameplay / Game Over)
