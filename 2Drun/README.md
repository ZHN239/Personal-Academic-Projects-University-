\# 🎮 Unity Side-Scrolling Runner (Personal)



\## 📌 Overview



A simple 2D side-scrolling platformer developed using Unity.

The game simulates continuous forward movement while the player controls jumping to navigate between platforms and collect items.



\---



\## 🎮 Gameplay



\* The character automatically progresses forward

\* Movement is simulated by shifting the environment to the left

\* Platforms are placed at different heights

\* The player must jump between platforms to avoid falling

\* Collectible stars are placed throughout the level



\---



\## 🚀 Features



\* Side-scrolling environment using camera/world movement

\* Jump mechanic controlled by the space key

\* Platform-based level design (multiple heights)

\* Collectible system (stars)

\* Basic collision detection



\---



\## 🛠️ Tech Stack



\* Engine: Unity

\* Language: C#



\---



\## 💡 Highlights



\* Implementation of side-scrolling mechanics using environment movement

\* Basic platformer physics and collision handling

\* Simple collectible system integration



\---

This project was an early exploration of Unity game development and 2D platformer mechanics.
