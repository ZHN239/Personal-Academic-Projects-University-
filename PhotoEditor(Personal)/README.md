\# Photo Editor



A fully functional Android photo management and editing application.



\## Features

\- 🖼️ Browse photos from your mobile album

\- ✏️ Photo doodling (supports multiple colors)

\- 🔄 Horizontal flip

\- ✂️ Center cropping

\- 💾 Save edited photos

\- 🗑️ Delete photos

\- 📅 Filter photos by date

\- 🌏 Multi-language support (Simplified Chinese / Traditional Chinese / English / Japanese)



\## Tech Stack

\- Java

\- Android SDK

\- RecyclerView

\- Glide

\- Multimedia storage operations
