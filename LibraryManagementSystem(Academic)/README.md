\# 📚 Full-Stack-Library-System (Academic)



\## 📌 Overview



A full-stack library management system consisting of a web application, an Android client, and a backend server.

The system supports keyword-based book search and allows users to manage a personal book list across platforms.



\---



\## 🚀 Features



\* Keyword-based book search

\* Add and manage books in a personal list

\* View detailed book information (title, price, ISBN, etc.)

\* Cross-platform access (Web \& Android)

\* Backend-driven data management



\---



\## 🏗️ System Architecture



\* \*\*Web Frontend\*\* → user interaction via browser

\* \*\*Android Client\*\* → mobile access

\* \*\*Backend Server\*\* → handles business logic and data processing

\* \*\*Database\*\* → stores book information
